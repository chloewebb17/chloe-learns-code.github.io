Code Log 1: HTML Semantic Tags

Semantic tags are HTML tags that clearly define their purpose to the browser and developer.

Non-semantic vs Semantic tags

Not all tags are semantic. If you want to make text thicker, you could use either the semantic <strong> tag, or the non-semantic <b> tag.

Strong actually indicates the function of the tag, but B is just a letter. As a designer myself, B seems a little more intuitive, but arguing about that would be... semantics (wink).

Generally, the idea is that non-semantic code needs much more context to be understood. Let's look at some examples of semantic and non-semantic tags now.
